Este projeto em Java tem como objetivo receber o nome de um aluno e duas notas informadas pelo usuário.
O sistema realiza o cálculo da média dessas notas e, com base no resultado, informa a situação do aluno.

Funcionalidades:
Entrada do nome do aluno
Entrada de duas notas
Cálculo da média
Exibição da situação do aluno:
Aprovado
Recuperação
Reprovado

Objetivo:
Este código foi desenvolvido para praticar lógica de programação, entrada de dados e estruturas condicionais na linguagem Java.
